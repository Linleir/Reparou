# Reparou React + Redux + json-server

## Como rodar

1. Extraia o zip.
2. Abra a pasta no VS Code.
3. No terminal, rode `npm install`.
4. Em um terminal, rode `npm run server`.
5. Em outro terminal, rode `npm run dev`.
6. Abra `http://localhost:5173`.
